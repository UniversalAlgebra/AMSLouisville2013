Special Session on Finite Universal Algebra

Fall Southeastern Sectional Meeting
University of Louisville, Louisville, KY
October 5-6, 2013 (Saturday - Sunday)
Meeting #1092

Room 215, Bingham Humanities, University of Louisville

At the AMS Southeastern Sectional meeting at the University of Louisville a special session on Finite Universal Algebra was organized by Ralph McKenzie and Matthew Valeriote.

You can download the abstracts and slides from the meeting to your local machine using the ZIP or TAR buttons on the left.

For Git users. If you want to suggest changes to this content, please use the GitHub button. Then fork the repository, make your changes, and submit a pull request. You can also clone the repository with the following command: 

git clone git@github.com:UniversalAlgebra/AMSLouisville2013.git

See also the official AMS special session program announcement.
