Special Session on Finite Universal Algebra

Room 215, Bingham Humanities, University of Louisville

At the AMS Southeastern Sectional meeting at the University of Louisville, October 5--6, 2013, a special session on Finite Universal Algebra was organized by Ralph McKenzie and Matthew Valeriote.

A repository containing slides for many of the talks has been created, and you can save these materials to your local machine using the ZIP or TAR buttons on the left.

For Git users. If you want to suggest changes to this content, please use the GitHub button. Then fork the repository, make your changes, and submit a pull request. You can also clone the repository with the following command: 

git clone git@github.com:UniversalAlgebra/AMSLouisville2013.git

For more information, see the official special session program announcement.
